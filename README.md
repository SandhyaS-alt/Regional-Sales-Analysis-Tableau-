# Regional Sales Analysis

In this project, I used Tableau to create a dashboard that 
visualizes the sales performance of two selected regions. This 
will help the organization understand and compare their 
performance and suggest necessary improvements. This project addressed the need for insightful 
sales analysis tools, enhancing proficiency in data analytics 
within the context of sales and performance analysis

## Key Questions Answered

1. Sales Performance Parameters
2. Average Sales and Order Statistics
3. Customer and Product Analysis
4. Comparative Analysis


## Highlights

* Dynamic Dashboard to visualize 2 region
* Enhanced Data visualization, storytelling and analytical skills.
* Mastered techniques like Data analysis and performance evaluation
